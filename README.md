# Web Server Project

This project is a simple **Node.js web server application** built to explore backend development concepts, routing, and dynamic web page rendering using **Express.js and EJS templates**.

The server delivers course-related pages and resources while handling user authentication and navigation between different views.

---

## 🚀 Features

- Node.js based backend server
- Dynamic web pages using **EJS templates**
- Static file serving (images, JS, etc.)
- Course data loaded from JSON
- User login and dashboard pages
- Multiple routes for course information and exams

---

## 🛠 Technologies Used

- **Node.js**
- **Express.js**
- **EJS (Embedded JavaScript Templates)**
- **JavaScript**
- **HTML / CSS**
- **JSON for course data**
