# Web Server Project

A Node.js web application built to explore backend development, authentication systems, AI-assisted learning features, and dynamic web page rendering.

The application provides course learning pages, AI-powered recommendations, exam generation, and a user dashboard. It uses Express.js, EJS templates, Firebase authentication, and AI integration to create an interactive learning platform.

---

## Features

- Node.js backend web server
- Dynamic pages using EJS templates
- Static file serving (images, JavaScript, etc.)
- Course content loaded from JSON files
- Firebase authentication for user login
- AI-assisted course recommendations
- AI-generated exam questions
- User dashboard for learning progress
- Course pages and exam pages

---

## Technologies Used

- Node.js
- Express.js
- EJS (Embedded JavaScript Templates)
- Firebase Authentication
- JavaScript
- HTML / CSS

## Installation

Clone the repository:

git clone https://github.com/Smdgameking/Web-server.git

Go into the project folder:

cd Web-server

Install dependencies:

npm install

---

## Running the Server

Start the server:

node server.js

Then open your browser and visit:

http://localhost:3000
